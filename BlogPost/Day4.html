Now let's get into the meat and potatoes of the code for the blog post. One first things that I changes with the blog posts is putting the actual post in the middle of the screen. I achieve this with the container class. Which can be any name as long as it has the display of grid. I just have to set up a div with a class and then any div that is a direct child of this class will be in the grid. Next to make the grid you need to have the attribute in your CSS of grid-template-columns or/and grid-template-rows. These attributes take in the width or height of amount of rows or columns you want. So if you want three columns you will need to put in three widths. Next thing styling the code and the post text. This is something where I ended up looking at a few different blog post to have an idea of how they did and what I thought looks good. The main thing that I got was I wanted to contain the code so it doesn't take up everything and make the text the star of the show. With the text I changed it to a paragraph element so that it would automatically wrap once it hit the end of its container. I also made a few changes to font size and added padding on sides. For the code I still wanted it to hold it's format so I left it as a pre element. Then I nested a code element so that the text would be styled the way I wanted it to. With the code since I wanted it only to take up small section I capped the height and made sure that overflow was set to scroll. This gave me the look that I wanted.
