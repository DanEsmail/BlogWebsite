The days are going by a lot faster than I can keep track of. We are already at day 18 and the site has come a long way. I still have a long way to go when you think about the 347 days left in the year. I finally took some time to make a skills page. Skills

Now I love this page as is but know there are a bunch of things that I can still do to improve. The main thing I wanted was to have a place that would almost experience bars and give me a sense that I was improving. Let's talk about some of the code that went into these.

Okay, time to break this down. First I wanted to have a title to get things going this would be followed by an image. I put them both in a div so I could control the container they were in. Most of of the images I got came from an icon finder and I attributed them all down at the button of my page. I used an H3 element to show the header title for each skill and added a picture element for the images. Next, we get to the stars and what “level” I am in the skill. This is all going to be in one row too so I put it within a div to contain it. I used a Style sheet to make the stars. It's called font awesome and I'll link the site below.

The final level of the skill box was the progress line. The way that I achieved this was to have a div nested within a div. Then I changed the color of the nested div. Once that was done I could use support classes to make the progress bar increase or decrease with the width percentage.

That is how I was able to create the skills page that you now see! I got a lot of work done and there is a ton more to get done. I want to learn how to use canvas for an upcoming fun project. I also have to learn some stuff about javascript objects and classes so be on the lookout for that.
